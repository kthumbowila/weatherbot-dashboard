# WeatherBot Free Dashboard

This is the first free web dashboard for the EE2323 WeatherBot project.

## Setup

1. Open `index.html` in a text editor.
2. Replace:
   - `PASTE_YOUR_SUPABASE_PROJECT_URL_HERE`
   - `PASTE_YOUR_SUPABASE_ANON_KEY_HERE`
3. Upload `index.html` to a GitHub repository.
4. Enable GitHub Pages from repository Settings → Pages.

## Free stack

- Supabase Free database
- GitHub Pages free hosting
- Plain HTML/CSS/JavaScript
- No React, no npm, no build tools
